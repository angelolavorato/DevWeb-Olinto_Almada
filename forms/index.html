Básico sobre Formulários

A tag <form> é usada para criar um formulário HTML para entrada de dados do usuário. Ela requer o atributo action para especificar para onde os dados do formulário serão enviados e o atributo method para definir como os dados serão enviados (por exemplo, GET ou POST).

Botões de Rádio e Caixas de Seleção

Botões de rádio permitem que os usuários selecionem uma opção de um grupo, enquanto caixas de seleção permitem múltiplas seleções.

Menus Dropdown e Áreas de Texto

Menus dropdown permitem que os usuários selecionem uma opção de uma lista, e áreas de texto são usadas para entrada de texto em várias linhas.



Acessibilidade e Validação

Sempre use rótulos (label) para entradas para melhorar a acessibilidade. Use atributos como required, min, max e pattern para validação.
